# AV1-CRUD
# Aplicativo CRUD Android em Java

Este repositório contém o código-fonte de um aplicativo CRUD (Criar, Ler, Atualizar, Deletar) simples desenvolvido em Java usando o Android Studio. O aplicativo permite que os usuários gerenciem entradas em um banco de dados, fornecendo uma interface amigável para adicionar, visualizar, atualizar e excluir registros.

## Funcionalidades

- **Criar**: Adicionar novos registros ao banco de dados.
- **Ler**: Visualizar e listar todos os registros no banco de dados.
- **Atualizar**: Modificar registros existentes no banco de dados.
- **Deletar**: Remover registros do banco de dados.
